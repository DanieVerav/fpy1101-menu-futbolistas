 

## Funcionalidades (CRUD)
Este proyecto incluye las siguientes funcionalidades a través de un menú interactivo:


- **Create**: Permite agregar un nuevo [futbolista] al sistema.
- **Read**: Muestra la lista completa de [futbolistas] registrados.
- **Update**: Modifica los datos de un [futbolistas] existente buscando por su nombre/ID.
- **Delete**: Elimina un [futbolista] del sistema.
