 

## Funcionalidades (CRUD)
Este proyecto incluye las siguientes funcionalidades a través de un menú interactivo:


- **Create**: Permite agregar un nuevo [futbolista] al sistema.
- **Read**: Muestra la lista completa de [futbolistas] registrados.
- **Update**: Modifica los datos de un [futbolistas] existente buscando por su nombre/ID.
- **Delete**: Elimina un [futbolista] del sistema.

## Requisitos Técnicos Aplicados
 el codigo en base de 4 def diferentes  que seria uno para agregar los futbolistas, eliminar los futbolistas,actualizar datos de los futblistas y ver los futbolistas agregados con su informacion correspondiente
 seguido con un while true para el menu que dara la opciones para ver,agregar,eliminar y actualizar para asi cumplir estos requisitos minimos 
- Uso de listas y diccionarios.
- Uso de funciones documentadas.
- Validaciones y manejo de excepciones (`try/except`).
